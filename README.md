Commands to run the project
# activates the virtual environment, command needs to be run in the project directory
.\venv\Scripts\activate

# run in the terminal
pyhton main.py

# I had issues in running the application with the above command, sometimes from Microsoft store you can run with this alternative
python3 main.py

# otherwise you can run directly avoiding any path issues with the command below
.env\Scripts\activate\python.exe main.py
